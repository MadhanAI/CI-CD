Quick demo

Try the Deploy tab to generate commands you can run on your EC2 server.

About

A tiny demo app to start from. You can extend it into a real tool that talks to servers or GitHub webhooks.

How to use

  1. Save this file as index.html and open in a browser.
  2. Customize UI or add a backend API to perform real SSH actions (be careful with private keys).
  3. For automation, connect GitHub webhooks to a secure server.

Deploy (GitHub → EC2) — Simulator

This simulates generating the SSH/git commands needed to pull your repo on EC2. It does NOT send keys or run commands from the browser.

Commands

No commands yet — click “Generate Commands”.
The generated commands are a guideline: connect via SSH using your .pem key, change to the repo folder, fetch and reset to the remote branch, then pull. Do not paste private keys into browser forms.
